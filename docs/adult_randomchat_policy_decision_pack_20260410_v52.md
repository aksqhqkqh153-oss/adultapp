# 어른플랫폼 랜덤채팅 정책 판단팩

## 이번 확정사항
- 차단 후 대화방은 완전 숨김
- 관리자 메시지 열람은 전체 상시 열람
- 랜덤채팅은 텍스트만 허용
- 메시지 삭제는 나와 상대방 모두 삭제 처리, 사용자 화면에는 `삭제된 메시지` 표기 유지
- 메시지 자가 삭제 허용 시간은 30분
- 재매칭 대기시간은 남성 20~40초, 여성 5~10초

## 허위 신고 처리 권장안
- 가장 무난한 방식은 **자동 누적점수 + 관리자 검토 혼합형**
- 권장 기준
  - 1~2점: 내부 누적만
  - 3점: 경고 플래그
  - 5점: 3일 랜덤채팅 제한
  - 8~10점: 7일 제한 + 관리자 검토 큐
  - 15점 이상: 장기정지/영구정지 수동 검토
- 자동 시스템은 `제한/검토대기`까지만, 최종 허위신고 확정은 관리자 판단 권장

## 재가입 우회 방지 권장안
- 최소: 이메일/아이디/전화번호 중복 제한
- 권장: 계정 + 디바이스 식별자 + 최근 IP/브라우저 지문 + 결제식별자 보조 비교
- 강한 수준: 재인증, 지연심사, 위험계정 수동승인

## 보존기간 분리 권장안
- 일반 대화: 30~90일
- 신고 접수 대화: 6개월~1년
- 제재 근거 증빙: 정지 해제 후 1년 내외
- 전자상거래 법정기록: 계약/청약철회 및 결제·재화공급 5년, 소비자 불만·분쟁처리 3년
- 개인정보는 목적 달성 후 지체 없이 파기, 법정기록은 일반 대화와 분리 보관

## 추가 결심 항목
- 신고 결과를 완전 비공개로 유지할지, 최소 상태값은 보여줄지
- 차단 해제 후 즉시 재노출할지, 일정 시간 제외할지
- 매칭 실패 시 자동 재탐색 횟수와 최대 대기시간
- 관리자 열람로그를 누가 언제 봤는지까지 남길지
- 랜덤채팅 전용 제재단계를 별도로 둘지
- 금칙어/외부연락 유도 필터 강도

## 이번 코드 반영 포인트
- 랜덤채팅 규칙 기본값 조정
- 차단 스레드 숨김 처리
- 메시지 전체 삭제 API 추가
- 랜덤채팅 종료 후 재매칭 쿨다운 구조 추가
