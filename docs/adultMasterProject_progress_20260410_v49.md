# adultMasterProject 진행 업데이트 v49 (2026-04-10)

- 랜덤채팅 정책을 운영 확정안 기준으로 정리했다.
- 거리 점수식은 band_bonus, 연령 인접 허용은 30대↔40대로 유지한다.
- 차단 해제는 사용자 자가 해제를 허용하되 관리자 로그는 항상 남긴다.
- 영구정지 사용자의 기존 채팅은 읽기전용으로 유지하고 관리자 전용 보관소에도 저장한다.
- 삭제 메시지는 사용자 화면에서 마스킹 문구로 표시하고 관리자 전용 보관소 원문은 유지한다.
- 지역 공개 단위는 시까지만 노출하도록 기준을 고정했다.
- 운영 DB는 Railway PostgreSQL 기준으로 통일하고 SQLite는 로컬 개발 전용으로 유지한다.
- 신고 자동정지 현재 스타터 기준은 5건/3일, 10건/7일, 20건/30일, 21건 이상 관리자 검토다.
- 다음 결심 과제는 고유 신고자 기준, 사유 가중치, 보관기간, 관리자 보관소 세부 권한, Redis 전환 시점 확정이다.
