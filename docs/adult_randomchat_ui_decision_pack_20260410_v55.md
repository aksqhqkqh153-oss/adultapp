# adultapp 랜덤채팅 UI/운영 결정 정리 v55

## 이번 반영 내용
- 성별 조건을 무관 / 남 / 여 / 기타로 구분
- 연령 조건을 20세~99세 최소/최대 바 형태로 변경
- 지역 조건이 거리기반일 때 0km~600km 최소/최대 바 형태로 변경
- 1:1랜덤채팅 매칭 성공 시 채팅방 자동 개설 및 채팅방 화면 자동 진입
- 채팅 > 랜덤 화면 상단에 시작 / 목록 카테고리 추가
- 목록 탭에서 개설된 1:1 랜덤채팅방을 목록화하고 남은 유지시간(분) 표시

## 현재 동작 기준
- 랜덤채팅은 텍스트 전용
- 메시지 삭제는 30분 이내 양측 삭제 표기 유지
- 신고 누적 제재는 랜덤채팅 기능만 24시간 / 72시간 / 7일 제한 우선
- 불법행위 / 사기유도 / 중대한 정책 위반은 서비스 전체 정지
- 차단/신고 시 대화방은 숨김 기준 유지

## 추가로 결심 필요한 항목
- 1:1 채팅방 유지시간을 30분으로 고정할지, 연장 기능을 둘지
- 랜덤채팅 목록에서 만료된 방을 즉시 제거할지, 최근 종료 목록으로 잠시 보일지
- 거리기반 매칭에서 최소/최대 거리 외에 현재 위치 오차 허용범위를 둘지
- 성별 기타 선택 시 노출 범위를 전체 허용으로 할지, 기타 포함 사용자만 우선 매칭할지
- 연령 바 선택 시 본인 연령과 너무 큰 차이가 나는 조건을 제한할지
- 랜덤채팅방에서 신고 버튼 위치를 상단 고정으로 둘지, 메뉴 내부로 둘지
- 랜덤채팅방에서 자동 종료 직전 5분 / 1분 알림을 줄지
- 관리자모드에서 랜덤채팅 필터값 수정 권한을 읽기 전용으로 둘지, 실수정 가능으로 둘지

## 권장 운영안
- 채팅방 유지시간은 기본 30분 + 필요 시 관리자 정책으로만 조정
- 만료 5분 전 알림 배너 표시
- 목록에서는 활성 방 / 종료 예정 / 만료 구분 배지 표시
- 신고/차단은 채팅방 헤더 우측 상단 고정
- 거리기반은 0~600km 범위를 유지하되, 실제 운영 전 위치 권한/오차 처리 규칙 별도 확정
