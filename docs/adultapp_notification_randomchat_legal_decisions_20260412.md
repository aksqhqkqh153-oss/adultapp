# adultapp 알림 버튼 · 홈 UI 클리핑 · 랜덤채팅 운영결정 문서 (2026-04-12)

## 1. 이번 반영 범위
- 우측 상단 검색 아이콘 오른쪽에 알림 버튼 추가
- 알림 화면을 공지 / 주문·배송 / 커뮤니티·채팅 3구역으로 분리
- 기존 하단의 이용약관 / 개인정보 처리방침 / 청소년 보호정책 / 환불정책 링크를 고정 하단에서 제거하고 알림 > 공지사항으로 이동
- 같은 정책 문구를 소통 > 커뮤 > 공지 카테고리 게시글에도 반영
- 하단의 `adultapp · 법인명 미정 / 사업자등록번호 미정 / 통신판매업 신고 미정` 문구 제거
- 홈 화면의 스토리/피드 카드가 레이아웃 밖으로 넘치지 않도록 overflow, min-width, contain, wrap 정책 보강
- 랜덤채팅은 고민상담 / 정보교류 목적의 텍스트 채팅으로 유지하되 사진/영상/파일 업로드는 영구 금지 방향 유지

## 2. 알림 버튼 UI 구조
### 상단
- 앱 공지사항
- 약관/정책 링크
- 앱 업데이트 공지

### 중단
- 쇼핑 주문 알림
- 배송 상태 알림
- 발송 진행 여부

### 하단
- 커뮤니티 댓글 알림
- 채팅 관련 운영기준 안내
- 기타 시스템 알림

## 3. 홈 화면 UI 클리핑 결정
첨부 HTML 구조 기준으로 아래 원칙을 적용한다.
- `.home-feed-pane`, `.home-story-card`, `.feed-post-list`, `.history-feed-card`는 모두 `max-width: 100%` 유지
- 스토리 스트립과 피드 리스트에 `overflow` 제어를 걸어 사진, 텍스트, 하위 카드가 부모 레이아웃 밖으로 이탈하지 않게 유지
- 텍스트가 길어져도 카드 폭을 밀어내지 않도록 `min-width: 0`, `overflow-wrap: anywhere`를 유지
- 피드 카드 내부 미디어 영역은 `overflow: hidden`으로 클리핑
- 카드 단위에는 `contain: layout paint`를 일부 적용해 하위 레이아웃 영향 범위를 줄임

## 4. 랜덤채팅 운영 방향
### 허용
- 고민상담
- 정보교류
- 제품 사용 경험
- 보관/세척/배송 관련 정보형 대화

### 금지
- 사진 업로드
- 영상 업로드
- 파일 업로드
- 전화번호 / 카카오톡 / 텔레그램 등 외부연락처 교환
- 오프라인 만남 유도
- 위치 공유
- 역할극·행위 제안

### 유지 결정
- 랜덤채팅 기능은 삭제하지 않음
- 단, "익명 정보교류용 텍스트 채팅"이라는 정책 문구를 유지
- 신고 + 차단 + 재매칭 제한 + 관리자/모더레이터 처리 권한 구조 유지

## 5. 제재 기준 확정안
- 1차 누적: 24시간 제한
- 2차 누적: 72시간 제한
- 3차 누적: 7일 제한
- 4차 이상 또는 중대위반: 영구정지

중대위반 예시
- 외부 연락처 유도
- 오프라인 만남 유도
- 노골적 성적 행위 제안
- 청소년 접근/유도 정황
- 반복 신고 + 명백한 회피 표현

## 6. 권한 결정
- 신고 열람: 관리자 / 모더레이터만
- 신고 처리(resolve): 관리자 / 모더레이터만
- 관리자 로그 조회: 관리자만
- 랜덤채팅 운영 로그/제재 이력 조회: 관리자만

## 7. 보존기간 판단 기준
현재 앱 방향성 기준 권장안
- 신고 / 제재 / 운영 로그: 3년
- 분쟁 / 환불 / 주문 관련 운영기록: 5년 범위 검토
- 랜덤채팅 일반 메시지 전문: 최소화 또는 짧은 기간 운영 후 파기 검토

결심 포인트
- 메시지 전문을 장기 보관할지
- 신고 발생 메시지만 별도 증거 보관할지
- 운영 로그와 결제/분쟁 로그를 분리 보관할지
