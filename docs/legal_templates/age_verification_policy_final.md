# 성인 인증 정책

- 버전: 2026-04-16.v1
- 시행일: 2026-04-16
- 상태: 운영 반영본

## 1. 기본 원칙
본 서비스의 쇼핑 영역은 성인용품 판매를 포함하므로 만 19세 이상만 접근할 수 있습니다. 회사는 PASS 기반 본인확인 또는 자체 성인 확인 절차를 통해 이용 가능 여부를 판단합니다.

## 2. 인증 수단
- PASS 또는 휴대폰 본인확인 연동
- 운영 테스트 단계에서는 자체 성인 확인(Self Certification) 보조 절차 사용 가능
- 최종 성인 인증 결과는 서버에서만 저장·판단합니다.

## 3. 미성년자 차단
- 만 19세 미만으로 확인되면 성인 전용 상품, 장바구니, 결제, 주문조회 접근이 차단됩니다.
- 우회 시도, 허위 입력, 반복 실패는 추가 제한 또는 계정 검토 대상이 됩니다.

## 4. 접근 제어
- 성인 인증 미완료: 안전모드 범위만 이용 가능
- 본인확인 완료 + 성인 인증 완료: 성인용품 상품 상세, 장바구니, 결제, 주문 기능 이용 가능
- 관리자 외 예외 허용 없음

## 5. 보관 항목
- 인증 수단
- 인증 시각
- 거래 추적용 TX ID
- 차단 또는 실패 이력

## 6. 고객 문의
성인 인증 오류 또는 이의 신청은 고객센터로 접수할 수 있습니다.
