# Adult Platform Random Chat Policy Decision Pack v53

## 1. 반영된 운영 결심
- 신고 결과는 사용자에게 별도 안내하지 않고 완전 비공개로 유지
- 차단 후 대화방은 완전 숨김 처리
- 차단 해제 후 상대는 즉시 다시 매칭 후보에 포함
- 랜덤채팅은 텍스트만 허용, 사진/영상/음성 메시지는 비활성 유지
- 메시지 자가 삭제는 30분 이내 허용, 삭제 시 나와 상대방 모두에게 `삭제된 메시지` 표기 유지
- 남성 재매칭 대기시간은 20~40초, 여성 재매칭 대기시간은 5~10초
- 매칭 실패 재시도는 4회 제한, 최대 탐색시간은 5분
- 관리자 전체 열람은 허용하되, 누가 어떤 스레드를 열람했는지 감사로그를 남김
- 외부 연락/오프라인/거래 유도 문구는 탐지 로직 없이 약관으로만 우선 관리
- 허위 신고 처리는 자동 누적점수 + 관리자 검토 혼합형
- 영구정지 재가입 우회 방지는 계정/전화번호/디바이스 식별자/최근 IP·브라우저 패턴 조합
- 영구정지 사용자는 1년 경과 후 재가입 재심사 가능

## 2. 질문 답변: 랜덤채팅 전용 별도 제재 단계란
의미는 `서비스 전체 이용정지`와 `랜덤채팅 기능만 제한`을 분리할지 여부입니다.

예시:
- 전체 서비스 정지: 로그인, 구매, 커뮤니티, 문의, 랜덤채팅 모두 제한
- 랜덤채팅 전용 정지: 로그인과 일반 구매/문의는 가능하지만 랜덤채팅 입장만 막음

### 권장안
- 운영 초반에는 **분리 운영**이 안전합니다.
- 사유가 랜덤채팅 내 욕설, 반복 신고, 무의미한 매칭 이탈, 허위 신고 누적이면 우선 `랜덤채팅 전용 제재`를 먼저 적용
- 심각한 불법행위, 미성년 의심, 사기/유도, 반복 악성행위는 `서비스 전체 제재`로 상향

### 권장 제재 단계
- 1단계: 랜덤채팅 24시간 제한
- 2단계: 랜덤채팅 72시간 제한
- 3단계: 랜덤채팅 7일 제한 + 관리자 검토
- 4단계: 서비스 전체 7일~30일 제한 또는 영구정지 검토

## 3. 허위 신고 처리 권장 기준
- 3점: 경고 플래그
- 5점: 3일 제한
- 8점: 7일 제한 + 관리자 검토
- 15점 이상: 장기정지/영구정지 검토

권장 운영 방식:
- 자동 점수 누적은 즉시 적용
- 실제 제재 확정은 관리자 검토와 병행
- 신고 결과는 비공개 유지
- 동일 상대 중복 신고는 1회만 허용

## 4. 보존기간 분리 권장안
- 일반 대화: 30~90일
- 신고 연계 대화: 6개월~1년
- 제재 증빙 및 재심 자료: 정지 해제 후 1년 내외
- 전자상거래 관련 법정기록: 관련 법정기간에 따라 별도 분리보관

운영 원칙:
- 목적 달성 후 지체 없이 파기
- 법정 보존 필요 기록은 운영 DB와 분리 보관
- 관리자 열람 로그는 신고/분쟁 대응 기준으로 함께 보관

## 5. 추가로 결심이 필요한 항목
### 5-1. 랜덤채팅 기능 정책
- 재매칭 쿨다운을 성별 외에 신고 이력/이탈 이력에도 가중할지
- 상대 차단 해제 직후 즉시 재노출을 유지할지, 24시간 제외 버퍼를 둘지
- 매칭 실패 4회 이후 안내문구를 보여줄지, 조용히 종료할지

### 5-2. 제재 정책
- 랜덤채팅 전용 제재와 전체 서비스 제재를 분리 운영할지 최종 확정
- 영구정지 1년 후 재가입은 자동 허용이 아니라 `재심 승인형`으로 할지 결정
- 재가입 시 과거 제재 이력을 새 계정에 연결할지 결정

### 5-3. 관리자 운영
- 관리자 스레드 열람 사유 입력을 강제할지
- 감사로그 다운로드 권한을 grade 1만 둘지
- 신고 검토 SLA 초과 시 관리자 알림을 넣을지

### 5-4. 약관/고지
- 외부 연락 유도 금지, 거래 유도 금지, 오프라인 만남 유도 금지를 약관에 명시
- 신고 결과 비공개, 관리자 열람 가능, 보관기간 분리를 개인정보처리방침에 명시
- 1년 후 재가입 재심 가능 조건을 운영정책에 명시

## 6. 이번 코드 반영 범위
- 랜덤채팅 정책 기본값 확장
- 랜덤채팅 전용 제재 정책 필드 추가
- 영구정지 재가입 대기기간 필드 추가
- 매칭 재시도 횟수/탐색시간 필드 추가
- 관리자 스레드 열람 감사로그 활성화 플래그 추가
- 텍스트 전용 / 결과 비공개 / 약관우선 관리 기본값 추가
