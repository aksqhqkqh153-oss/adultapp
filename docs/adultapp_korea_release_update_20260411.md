# adultapp 국내 출시 법적 보완 업데이트

- 버전: 2026-04-11.v1

## 반영한 내용
- 사업자/통신판매/청소년보호책임자 등 국내 출시 필수 표시정보 설정값 추가
- `/api/legal/business-info` API 추가
- `/api/legal/release-readiness` API 추가
- 프론트 상단에 국내 출시 보완 배너 추가
- 하단 고정 법정 링크에 환불정책 추가
- 하단 사업자 표시정보 푸터 추가
- 개인정보처리방침/이용약관/청소년보호정책/환불정책 문구 보강
- 국내 출시 전 법적 차단 요소 문서 추가

## 이번 반영의 목적
- 한국 출시 직전 가장 많이 빠지는 표시의무/운영연동/문서 고지 누락을 앱 안에서 바로 보이게 만들기
- 출시 차단 항목과 주의 항목을 운영자가 즉시 확인하도록 만들기

## 아직 남은 핵심
- PASS 운영 연동 실키 적용
- PG 운영키/환불 webhook 적용
- 사업자/통신판매업/청소년보호책임자 실제 정보 입력
- 위치/지역 기반 기능 고지 확정
- 미성년 차단 계정 자동 파기 배치
