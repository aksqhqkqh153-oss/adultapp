# adultapp shop business verification / product register update (2026-04-12)

## 적용 내용
- 쇼핑 상단바의 `등록관리` 탭을 `사업자인증`, `상품등록`으로 분리했다.
- 쇼핑 목록 화면의 `상품등록` 버튼 클릭 시 사업자 미인증 계정은 자동으로 `사업자인증` 화면으로 이동하도록 변경했다.
- `사업자인증` 화면에서 사업자등록 정보, 정산 정보, 인증 사진 URL을 입력/수정하고 인증 신청할 수 있게 유지했다.
- `사업자인증` 화면은 쇼핑 메뉴에서 다시 진입해 수정할 수 있다.
- `상품등록` 화면은 사업자 승인 계정 또는 관리자 계정만 실제 등록 가능하도록 유지했다.
- 관리자 계정은 사업자 인증 없이도 상품 등록 가능하며, 등록 직후 자동 승인 처리되어 일반 회원 쇼핑 목록에도 바로 공개되도록 연결했다.

## 테스트 권장 흐름
1. 관리자 계정 로그인
2. 쇼핑 > 상품등록 진입
3. 상품 입력 후 임시저장
4. 쇼핑 > 목록에서 일반 회원 기준 공개 여부 확인
5. 회원 계정 로그인
6. 쇼핑 > 목록에서 관리자 등록 상품 확인
7. 장바구니 > 주문 > 결제/취소/부분취소/환불/webhook 점검 진행

- 판매자: `seller@example.com` / `seller1234`
- 일반회원: `general@example.com` / `general1234`
