# adultapp 서비스 준비도 보완 후 추가 의사결정 항목

## 1. 성인인증 공급사 선택
- 결심 필요: PASS 단독 / 통합인증 / NICE·Danal 등 어떤 공급사를 실제로 계약할지
- 추천: PASS 또는 휴대폰 본인확인 중심, 카카오는 로그인 편의용 유지

## 2. 미성년 처리 정책
- 결심 필요: 계정 전체 차단인지, 성인 영역만 차단인지
- 추천: 계정 유지 + 성인 영역 전체 차단

## 3. 쇼핑 구조
- 결심 필요: 직접 판매형인지, 중개 플랫폼형인지
- 영향: 세금, 환불 책임, 판매자 계약, PG 심사 문구가 달라짐

## 4. 판매자 검증 수준
- 결심 필요: 사업자등록증 + 정산계좌 + 반품지 + CS연락처까지 필수인지
- 추천: 모두 필수화 후 검증 완료 전 판매 제한

## 5. 재동의 정책
- 결심 필요: 약관/처리방침 버전 변경 시 강제 재동의 범위
- 추천: 필수 문서만 강제 재동의, 선택 동의는 유지

## 6. 데이터 보관 정책
- 결심 필요: 탈퇴 즉시 삭제 vs 법정 보관 + 운영 로그 유지
- 추천: 결제/환불/보안/신고 이력은 법정/운영 필요 범위 보관, 나머지는 삭제

## 7. 채팅 운영 강도
- 결심 필요: 신고 중심인지, 자동 차단 강도를 높일지
- 추천: 연락처 유도/불법 권유/반복 욕설은 강한 자동 제재

## 8. 앱 심사 노출 구조
- 결심 필요: 안전모드 고정 홈을 별도 운영할지
- 추천: 심사용 safe 홈/검색/쇼핑 화면 유지
