# 어른플랫폼 랜덤채팅 정책/관리자모드 업데이트 v54

## 이번 반영 요약
- 랜덤채팅 신고 누적 제재를 서비스 전체 정지와 분리해 랜덤채팅만 24시간/72시간/7일 제한하도록 유지
- 불법행위, 사기 유도, 중대한 정책 위반은 서비스 전체 정지 대상으로 유지
- 설정 화면에 관리자 전용 `관리자모드` 카테고리 추가
- 관리자모드 하위에 `DB관리 / 신고 / 채팅 / 기타` 탭 추가
- 관리자 화면에서 현재 랜덤채팅 기준, 신고 현황, 채팅 스레드 현황, 관리자 감사로그를 조회 가능하도록 연결

## 이번 관리자모드 구조
### 1) DB관리
- 신고 DB 총 건수와 상태 집계
- 채팅 DB 총 스레드 수와 상태 집계
- 랜덤채팅 전용 제재 기준 확인
- 관리자 열람 범위 및 감사로그 활성 여부 확인

### 2) 신고
- 신고 결과 완전 비공개 유지 여부 확인
- 최근 신고 목록 확인
- 최근 신고 사유 집계 확인
- 랜덤채팅 전용 제재 기준 문자열 확인

### 3) 채팅
- 차단 후 대화방 완전 숨김 정책 확인
- 메시지 삭제 범위(나와 상대 모두 삭제 + 삭제된 메시지 표기 유지) 확인
- 재매칭 재시도 4회 / 최대 탐색 5분 확인
- 최근 랜덤채팅 스레드 목록 확인

### 4) 기타
- 관리자 전체 열람 감사로그 최근 기록 확인
- 영구정지 후 1년 뒤 재가입 재심사 기준 확인
- 메시지 자가 삭제 30분 기준 확인

## 추가 결심이 필요한 항목
- 랜덤채팅 전용 24시간/72시간/7일 제한 이후에도 반복 위반 시 전체 서비스 제재로 올리는 승급 규칙 확정
- 관리자 감사로그 보관기간 확정
- 관리자모드에서 실제 수정 기능까지 열지, 현재처럼 조회 중심으로 둘지 확정
- 향후 이미지/음성/영상 오픈 시 별도 검수 큐와 보관 정책 추가 여부 확정
- 약관/개인정보처리방침/운영정책 문구를 현재 시스템 기준과 동일하게 맞출지 확정

## 질문 답변
### 랜덤채팅 전용 제재 단계가 무슨 뜻인가
전체 서비스 이용은 가능하지만 랜덤채팅 기능만 따로 제한하는 방식입니다.
예: 욕설/허위신고/반복 이탈은 랜덤채팅만 24시간 또는 72시간 제한, 구매/문의/커뮤니티는 유지.
반대로 불법행위/사기 유도/중대한 정책 위반은 전체 서비스 정지로 올립니다.
