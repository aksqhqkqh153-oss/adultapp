# adultapp 수익화 방향 업데이트 2026-04-12

## 반영 방향
- 판매자 월 구독제는 사용하지 않는다.
- 홈 피드/질문 피드 사이에 검수형 추천노출 상품 슬롯을 둔다.
- 구매자 회원제 기반 프리미엄 배송 혜택을 둔다.
- 판매자 대상 B2B 리포트/대시보드/증빙 기능은 유료화한다.

## 앱 내 소통 활성화 10가지
1. 안전수칙 요약 토론방
2. 동의와 경계설정 체크인
3. 초보 입문 Q&A
4. 익명 고민상담 게시판
5. 일상/취미 라운지
6. 제품 사용/보관 팁 교류
7. 구매 전 질문 스레드
8. 후기형 짧은 댓글 토론
9. 운영진 진행형 AMA
10. 주간 주제 토크방

## 금지 기준
- 오프라인 만남 제안
- 외부 연락처 교환
- 사진/영상/파일 전송
- 성매매/대가성 제안
- 지역/거리 기반 매칭

## 판매자 B2B 유료 기능
- 월별 정산 리포트
- 반품/환불 이력 리포트
- 판매자 분쟁 대응 로그
- 증빙 요청/다운로드 기능
- 사업자용 대시보드
- SKU 승인 상태 리포트
