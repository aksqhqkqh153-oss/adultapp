# adultapp marketplace policy decisions update 2026-04-11

## 반영한 정책
- 판매자 승인 기준: 사업자등록증, 정산계좌 확인, 반품지, CS 연락처, 판매자 약관 동의까지 완료해야 승인
- 상품 승인 정책: 승인 전 비공개, 사업자 계정 본인에게만 승인대기 상태 노출, 관리자 승인 후 공개
- 세금계산서/현금영수증 주체: 직접판매는 운영사, 중개판매는 판매자 발급 원칙
- 정산 주기: 초기에는 환불/CS 안정화 후 주별 정산
- 백오피스 우선순위: 판매자 승인 상세 화면 우선, 정산 CSV/월마감은 다음 단계

## 코드 반영
- 관리자모드 > 승인 탭 추가
- 관리자모드 > 정산 탭 추가
- 판매자 승인 큐 / 상세 / 승인·보류·반려 API 추가
- 상품 승인 큐 / 상세 / 공개 승인·보류·반려 API 추가
- 사업자 인증 신청 시 정산계좌 정보 저장
- 판매자 본인 상품 목록 조회 API 추가
- 상품 임시저장 후 승인대기 제출 API 추가
- 상품 공개 전 상태는 draft/pending_review/rejected/approved 로 관리
- 공개 상품 목록은 approved 상태만 노출
- 정산 미리보기 API에 세금계산서/현금영수증 발급 책임, 정산 주기 정책 포함
