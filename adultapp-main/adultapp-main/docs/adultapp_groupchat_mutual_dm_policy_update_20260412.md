# adultapp 그룹대화 + 상호수락형 1:1 정책 업데이트 (2026-04-12)

## 이번 반영 요약
- 공개 앱에서 성향/관심사 기반 **주제형 그룹대화**를 허용
- 1:1 대화는 **그룹대화 참여 후 상호 수락형**으로만 허용
- 자유 개시형 DM, 외부 연락처 교환, 오프라인 제안, 사진/영상 전송은 금지 유지
- 방 개설은 운영자만 허용하고 주제형 카테고리만 사용

## 허용 카테고리
- 관계역할/고민
- 동의/경계설정
- 안전수칙
- 일상/취미
- 자유대화

## 금지 항목
- 파트너 모집
- 만남 유도 및 오프라인 제안
- 외부 연락처 교환
- 사진/영상/파일 전송
- 역할극/행위 제안
- 대가성 제안
- 미성년/비동의/강압 표현

## 1:1 허용 조건
- 성인인증 완료
- 최근 신고/제재 이력 없음
- 그룹대화 참여 이력 존재
- 상호 수락 완료
- 감사로그 저장 동의

## 운영 판단
직접적인 자유 DM 개방은 법률·스토어 심사·운영 리스크가 높아 이번 반영에서는 제외했다.
대신 성향/관심사 기반 소통 수요는 그룹대화와 상호수락형 1:1 구조로 흡수하도록 설계했다.
