# 국내 앱 출시 전 법적 보완 항목

- 버전: 2026-04-11.v1

## 가장 큰 차단 요소
1. PASS/휴대폰 본인확인 운영 연동 미완료
2. PG 운영키/환불 webhook 미완료
3. 사업자/통신판매업/청소년보호책임자 표시 정보 미확정
4. 위치·지역 기반 기능의 고지 문구 및 동의 구조 미정
5. 미성년 차단 계정 자동 파기 배치 미완료

## 권장 조치
- 공개 출시 전에는 운영키와 webhook 검증을 완료
- 사업자 고지 정보 확정 전까지 스토어 제출용 설명과 앱 내부 표시를 임시 문구로 두지 말고 미입력 항목을 실제 값으로 교체
- 랜덤채팅/거리기반 기능은 위치·지역 정보 안내 문구를 회원가입/프로필/기능 진입 전 단계에 추가
- 쇼핑은 판매자 인증 + 관리자 사전 승인 구조 유지
- 재동의 유예 종료 후 제한 기능을 서버와 프론트에 동일하게 적용

## 사용자 판단 필요
- 재동의 제한 모드를 limited_access로 유지할지 login_block으로 강화할지
- 통신판매 주체를 직접판매로 할지 중개로 할지
- 위치기반 기능을 실제 GPS 기반으로 확대할지 지역 선택형으로 제한할지
- 공개 베타 전에 상품 공개를 관리자 승인형으로 유지할지
