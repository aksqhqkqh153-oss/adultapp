# adultapp 아라곤왕국 정책 비교 및 안전 반영안 (2026-04-12)

## 목적
아라곤왕국 운영규칙을 참고하되, 공개 앱에서의 법적·심사 리스크를 낮추기 위해 어떤 요소는 반영하고 어떤 요소는 제거해야 하는지 운영 기준으로 정리한다.

## 1. 참고 규칙에서 상대적으로 안전한 요소
- 만 19세 이상 성인만 가입하도록 제한
- 선정적 나체, 음란행위 표현, 미성년 성적 묘사 금지
- 성매매, 일회성 플레이, 혼음·관전, 불륜·스와핑 구인 금지
- 반복 접촉, 스토킹, 협박, 개인정보 유출, 사진 무단 전재 금지
- 신고·탐문·조사·가중처벌·영구정지 구조 운영
- 오프라인 모임에서 숙박 전제, 음란행위 목적, 성을 매개로 한 장소 변질 금지

## 2. adultapp에 그대로 넣으면 위험한 요소
- 친구찾기/인연찾기 게시판
- 쪽지 및 1:1 채팅을 일반 회원에게 폭넓게 허용하는 구조
- 모임소식 게시판 및 회원 주최 오프라인 모임
- 포인트를 소통 기능과 결합하는 구조
- 비밀글 예외를 통해 외부 연락처 제공 여지를 남기는 구조
- 게시판 금지와 별도로 DM/1:1 채팅에서 사실상 구인이 이뤄질 수 있는 구조

## 3. adultapp 최종 반영 방향
### 앱 공개영역에 유지
- 쇼핑
- 주문/결제/배송
- 판매자센터
- 운영공지
- 제품 정보 Q&A
- 세척/보관/환불/익명포장 정보형 커뮤니티
- 운영문의/판매자응답 채팅

### 앱 공개영역에서 제거 또는 기본 비활성화
- 랜덤채팅
- 사용자간 자유 DM
- 친구/인연찾기
- 모임 개설/모임소식
- 오프라인 만남 유도 기능
- 외부 연락처 교환 허용 구조
- 사진/영상 기반 민감 커뮤니티 기능

### 제한 웹 영역으로만 분리 검토
- 성인인증 + 운영정책 동의 + 승인제 심사를 통과한 회원 전용 포럼
- 텍스트 중심 정보교류
- 허용 주제: 안전수칙, 동의/경계설정, 세척/보관, 배송/환불 경험, 제품 정보
- 금지 주제: 파트너 모집, 만남 유도, 외부 연락처 교환, 사진/영상 공유, 역할극/행위 제안

## 4. 운영 원칙
- 공개 앱은 `커머스 + 정보형 커뮤니티`로 고정
- 민감 소통은 앱 밖 제한 웹으로 분리
- 앱 내에서는 운영문의/판매자응답 채팅만 허용
- 제한 웹은 이미지 업로드 기본 금지, 신고/차단/감사로그 의무화
- 운영진 승인이 없는 회원은 읽기 전용 또는 미입장 상태 유지

## 5. 이번 코드/정책 반영
- 랜덤채팅 API 진입을 안전모드 기본값으로 차단
- 공개 앱 UI에서 랜덤 매칭 대신 제한 웹 포럼 분리 운영 안내로 교체
- 릴리즈 준비도 점검에 고위험 소셜 기능 비활성화 항목 추가
- 운영자가 판단해야 할 결심 항목 문서를 별도로 추가
