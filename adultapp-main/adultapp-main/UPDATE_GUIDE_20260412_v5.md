# adultapp 로그인 401/404 완화 업데이트

## 반영 내용
- 저장된 만료 토큰이 있을 때 `/auth/me` 401이 반복되는 문제 완화
- 토큰이 없으면 앱 시작 시 `/auth/me`를 호출하지 않도록 변경
- `/auth/me`가 401이면 자동으로 토큰 정리 후 GUEST 상태로 전환
- `/legal/business-info`, `/legal/release-readiness`는 관리자 인증 후에만 조회하도록 변경
- 일반 로그인/테스트 계정 로그인 시 기존 토큰을 먼저 정리하도록 보강
- 로그인 실패 시 토큰 잔존 상태를 정리하도록 보강

## 사용자 체감 효과
- 로그인 화면 첫 진입에서 401/404 콘솔 오류 감소
- 만료 토큰 때문에 로그인 화면이 꼬이는 현상 감소
- 관리자 전용 legal API를 비로그인/일반회원에서 호출하지 않음
