# adultapp header favorites / forum menu update

## 변경사항 요약
- 상단바 메뉴 오버레이 상단에 `상단바 즐겨찾기` 편집 영역 추가
- 홈/쇼핑/소통/채팅/프로필별 상단 버튼을 추가/제거하고 로컬 저장소에 저장
- `채팅 > 포럼` 내용을 `소통 > 포럼`으로 이동
- `채팅 > 채팅` 화면의 `1:1 채팅 운영 기준` 고정 문구 제거
- 단체 톡방 안내를 `소통 > 포럼`으로 이동

## 법적/운영상 추가 판단 필요 항목
1. `성향 고민` 버튼 문구 유지 여부
   - 앱 심사/민원 대응에서는 민감하게 읽힐 수 있으므로 `관계 고민` 또는 `역할 고민`으로 더 중립화할지 최종 결정 필요
2. 단체 톡방 개설 권한
   - 현재 데모는 개설 버튼이 열려 있으므로, 실운영에서는 운영자 승인형 또는 성인인증+무신고 계정만 허용할지 결정 필요
3. 포럼 참여 후 1:1 조건
   - 현재 구조는 상호 팔로우 + 요청/수락형 전제이므로, 추가로 가입 후 유예기간/누적 신고 기준을 더 넣을지 결정 필요
4. 계정비공개 범위
   - 현재 프론트 안내 기준은 상호 팔로잉 외 비공개이며, 실제 배포 전에는 백엔드 API에서 프로필/피드/채팅 접근제어까지 동일하게 막을지 결정 필요
5. 단체 톡방 허용 범위
   - 정보 교류형/고민형/제품형 주제방만 허용하고, 오프라인 모임/외부 연락처/사진·영상/대가성 제안은 계속 금지하는 방향 권장

## 질문 답변 메모
- `채팅 > 단체톡방` 형태의 고민/정보 교류 방은 가능성은 있으나, 주제형 정보교류 방으로 운영하고 외부 연락처, 사진/영상, 오프라인 제안, 대가성 제안, 미성년/비동의/강압 표현을 차단하는 전제가 필요
