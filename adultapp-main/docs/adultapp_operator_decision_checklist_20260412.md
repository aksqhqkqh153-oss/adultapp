# adultapp 운영자 결심 체크리스트 (2026-04-12)

## 1. 서비스 구조 결심
- 공개 앱을 `성인용품 중개 거래 + 정보형 커뮤니티`로 고정할지
- 민감 소통 기능을 앱에서 완전히 뺄지
- 제한 웹 포럼을 별도 서브도메인 또는 별도 서비스로 둘지

## 2. 고위험 기능 결심
- 사용자간 자유 DM을 완전 금지할지, 판매자/운영문의만 열지
- 랜덤채팅을 영구 제거할지, 내부 테스트만 유지할지
- 모임/친구찾기/인연찾기 기능을 전부 포기할지
- 외부 연락처 교환 적발 시 즉시 영구정지 또는 누적 제재로 갈지

## 3. 법적·운영 정보 확정
- 법인명 / 사업자등록번호 / 통신판매업 신고번호
- 사업장 주소 / CS 전화 / 이메일
- 청소년보호책임자 실명
- 반품지 / 분쟁처리 연락 채널
- 직접판매와 중개판매를 함께 할지 분리할지

## 4. 본인확인/성인인증 결심
- PortOne PASS 운영키 발급 후 공개 전환 여부
- 미성년 차단 이력 보관 기간
- 본인확인 결과를 최소 토큰만 저장할지, 상세 로그를 더 남길지
- 재인증 기준(주기, 실패 횟수, 쿨다운)

## 5. 커뮤니티 운영 결심
- 이미지 업로드를 전면 금지할지
- 제한 웹 포럼에서 댓글만 허용할지, 글쓰기까지 승인제로 열지
- 신고 처리 SLA와 담당자 역할 분리를 어떻게 할지
- 운영자 열람 범위와 감사로그 보관 정책을 어디까지 둘지

## 6. 상품/결제 결심
- 허용/제한/금지 SKU 표를 어디까지 보수적으로 시작할지
- PG 승인 가능 상품군만 먼저 오픈할지
- 환불 제한 문안과 반품 불가 조건을 어디까지 둘지
- 세금계산서/현금영수증 책임 주체를 직접판매/중개판매별로 확정할지

## 7. 앱 심사 결심
- 앱은 쇼핑/주문/문의/정보형 커뮤니티까지만 둘지
- 제한 웹 포럼 링크를 앱 내부에 노출할지, 외부 공지로만 안내할지
- 스토어 스크린샷/설명에서 민감 표현을 전부 제거할지

## 8. 오픈 전 최소 완료 조건
- 성인인증 실연동
- PG 승인/취소/환불 webhook 점검
- 사업자 고지 실정보 입력
- 약관/처리방침/청소년 보호정책 최종본
- 판매자 승인/상품 승인/신고처리 백오피스 점검
- Cloudflare Pages 수동 배포 및 캐시 재확인
