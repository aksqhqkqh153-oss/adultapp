# adultapp API/policy/random chat update - 2026-04-12

## 이번 반영 요약
- 삭제/업로드/주문/정산/신고/관리자로그/세무 다운로드 API의 접근 제어를 실운영 기준으로 보강했다.
- 커뮤니티를 정보형 범위로 제한하고 외부 연락처 유도, 오프라인 만남, 사진 교환, 후기 위장 유도 문구를 텍스트 필터에 추가했다.
- 업로드 API에 로그인 필수, 확장자/MIME 검사, 악성 시그니처 차단, presigned 구조 스텁 응답을 추가했다.
- 월마감 CSV/PDF, 관리자 액션 로그는 관리자만 접근 가능하게 고정했다.
- 신고 목록/처리는 관리자·서브관리자·매니저만 접근 가능하게 제한했다.

## 코드 반영 상세
### 1) 권한 제어
- 주문 목록 조회: 본인 주문 또는 해당 셀러, 관리자/운영자만 조회 가능
- 정산 미리보기: 관리자/운영자 또는 해당 셀러만 조회 가능
- 신고 목록/처리: 관리자/서브관리자/매니저만 가능
- 관리자 액션 로그: 관리자만 가능
- 세무 월마감 CSV/PDF: 관리자만 가능
- 상품 삭제: 작성 셀러 또는 관리자만 가능
- 콘텐츠 삭제/수정: 작성자 또는 관리자만 가능
- 상품 미디어 첨부: 해당 상품 셀러 또는 관리자만 가능

### 2) 커뮤니티/채팅 필터
- 외부 연락처 유도 금지
- 오프라인 만남/숙소/주소 교환 유도 금지
- 역할극/노출/신체사진/영상 교환 유도 금지
- 판매 후기처럼 보이게 하며 개인 연락을 유도하는 문구 금지
- 반복적 사적 접촉 유도 문구 금지
- 허용 카테고리 외 게시글 차단

### 3) 업로드 API
- 로그인 필수
- 허용 확장자만 허용: png/jpg/jpeg/webp/mp4/mov/webm
- MIME mismatch 차단
- 실행파일/스크립트/압축파일 업로드 차단
- 기본 악성 시그니처(MZ, script 태그) 차단
- presigned 업로드 구조 스텁 반환

## 아직 남는 실운영 결정/외부 연동 항목
### 반드시 직접 확정/준비가 필요한 항목
1. 실제 사업자 정보
- 법인명/사업자번호/주소/대표 CS 번호/개인정보 문의 메일
- 통신판매업 신고번호
- 청소년보호책임자 성명
- 반품지/고객센터 연락처

2. 성인인증 실연동
- PortOne PASS 운영 store id / channel key 발급
- callback/webhook 운영 시크릿 발급
- 실패/미성년 차단/재시도 정책 확정
- 본인확인 결과 저장 범위와 보관기간 확정

3. PG/결제 실연동
- 승인/취소/환불 webhook 실제 서명값 연결
- 실 결제수단 분기 및 PG 승인 가능한 상품군 재분류
- 세금계산서/현금영수증 발급 책임 주체 최종 확정

4. 문서 최종본
- 개인정보처리방침 최종본
- 판매자 약관
- 정산 정책
- 입점 계약서
- 환불/반품 제한 문안 최종본

## 랜덤 채팅 기능을 법적 리스크 낮게 넣는 기준
### 권장 구조
- 목적을 “익명 정보교류”로 제한
- 실명/전화번호/외부 메신저/오프라인 만남 유도 전면 금지
- 사진/영상 업로드 차단 또는 강한 사전검수 적용
- 신고/차단/재매칭 제한/대화종료/감사로그 의무화
- 19세 이상 + 본인확인 완료 사용자만 입장
- 첫 화면과 매칭 전 화면에 금지행위 고지 및 재동의 노출
- 운영자 열람 권한, 신고 SLA, 제재 정책 명문화

### 피해야 하는 구조
- 파트너 모집, 만남 주선, 외부 연락 교환 유도
- 성적 역할극/행위 제안 중심의 매칭 UX
- 위치 실시간 공유, 근처 사용자 즉시 연결
- 신고 후 제재/증적이 남지 않는 구조

### 서비스 문구 방향
- “랜덤 만남” 금지
- “익명 안전 정보교류” 권장
- “제품 사용/안전/보관/후기 관련 정보교류” 중심으로 한정

## 출시 가능 여부 판단
### 현재 상태
- 개발 스타터/운영 가드레일 수준까지는 접근했지만, 법무/PG/본인확인 운영키/사업자 실정보가 미정인 상태라 즉시 정식 출시 가능 상태는 아님.

### 출시 전 최소 완료 조건
- 사업자/통신판매중개 고지 실정보 반영
- PortOne PASS 운영 연동 및 webhook 검증 실테스트
- PG 승인/취소/환불 webhook 실데이터 점검
- 약관/개인정보/청소년보호/환불 문서 최종본 반영
- 관리자 신고 처리/감사로그/월마감 백오피스 운영 테스트

## 결심이 필요한 항목
1. 랜덤 채팅의 범위를 정보형으로만 제한할지 여부
2. 사진/영상 업로드를 랜덤 채팅에서 완전 금지할지 여부
3. 외부 연락처 교환 적발 시 즉시 1회 차단인지, 누적 제재인지 여부
4. 실명 사업자 고지 범위를 어디까지 노출할지 여부
5. 직접판매와 중개판매를 한 앱에서 같이 운영할지, 탭/도메인/법인 분리할지 여부
6. 성인인증 결과 보관 범위(최소값 저장 vs 상세로그 저장) 결정
7. 판매 후기 허용 범위를 배송/포장/환불 중심으로 제한할지 여부
