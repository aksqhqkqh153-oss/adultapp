# 어른플랫폼 랜덤채팅 정책 판단 정리 v51

## 이번 반영 핵심
- 동일 사용자는 같은 상대방을 랜덤채팅에서 1회만 신고 가능
- 신고 즉시 신고자 계정 기준으로 상대 계정 자동 차단
- 차단 직후 즉시 해제 허용
- 랜덤채팅은 텍스트만 우선 허용
- 신고 처리 결과는 별도 사용자 안내 없이 내부 처리
- 관리자 보관소 및 신고관리 권한은 현재 grade 1 전용
- 개인정보는 목적 달성 후 지체 없이 파기하되, 법정 보존기록은 분리 보관
- 전자상거래 기록 보존은 계약/청약철회 5년, 대금결제·재화공급 5년, 소비자 불만·분쟁처리 3년 기준

## “사용자 자가 삭제 허용 시간”의 의미
이 항목은 보통 **사용자가 본인이 보낸 채팅 메시지를 스스로 삭제할 수 있는 제한 시간**을 뜻합니다.
예: 메시지 전송 후 5분 이내에는 본인만 삭제 가능, 5분이 지나면 일반 사용자는 삭제 불가, 관리자 보관소에는 마스킹 상태로 보관.

### 권장 기본값
- 권장값: **5분**
- 이유: 오발송 수정은 가능하게 하면서, 대화 왜곡과 증거 인멸 리스크는 과도하게 키우지 않음

### 시간대별 운영 판단
- 1분 이하: 너무 짧아서 오타/오발송 정정 체감이 낮음
- 5분: 가장 무난한 기본값
- 10분: 사용성은 좋아지지만 분쟁 시 문맥 훼손 가능성이 조금 커짐
- 30분 이상: 신고·분쟁 대응에서 증거 보전 리스크가 커짐

## 추가로 확정이 필요한 판단 항목

### 1) 메시지 삭제 범위
- 판단할 것: “나에게만 삭제”인지, “상대 화면에서도 삭제”인지
- 권장: **상대 화면에서는 ‘삭제된 메시지’로 마스킹 표시**, 관리자 보관소에는 원문 보관

### 2) 신고 사유 코드 고정 여부
- 판단할 것: 신고 항목을 고정할지, 자유입력을 크게 열지
- 권장: **고정 사유 + 상세설명 선택형**
- 기본 사유: 욕설, 스팸, 개인정보요구, 음란물전송, 불법권유, 기타

### 3) 허위 신고 제재 기준
- 판단할 것: 허위 신고 누적 기준과 단계
- 권장 기준: **3점 경고 / 5점 3일 정지 / 10점 7일 정지 / 15점 관리자 검토**
- 운영 원칙: ‘기각’만으로 바로 허위신고 처리하지 말고, **악의성 또는 반복성 확인 시에만 누적**

### 4) 차단 후 스레드 보존 범위
- 판단할 것: 차단 후 대화방을 완전히 숨길지, 읽기전용으로 남길지
- 권장: **당사자에게는 읽기전용 또는 목록 잔존, 재연결 금지**
- 이유: 신고 직후 사실관계 확인과 자기 보호 기록 확인에 유리

### 5) 관리자 메시지 열람 범위
- 판단할 것: grade 1이 전체 원문을 항상 볼지, 신고/분쟁건만 볼지
- 권장: **기본은 신고·분쟁·법적 요청 건 중심 접근 + 관리자 로그 필수**
- 현재 인력 기준으로는 grade 1 전용 유지가 적절

### 6) 이미지/파일 전송 허용 시점
- 판단할 것: 언제 텍스트 외 매체를 열지
- 권장: **초기에는 텍스트만**, 이후 신고율·차단율·운영 여력 확인 후 단계 오픈
- 순서 권장: 텍스트 → 이미지(자동블러/신고강화) → 파일/음성은 보류

### 7) 랜덤채팅 매칭 해제/재매칭 정책
- 판단할 것: 매칭 직후 이탈 시 바로 재매칭 허용 여부
- 권장: **즉시 종료는 허용하되, 짧은 쿨다운(예: 10~30초) 부여**
- 이유: 매칭 남용과 무의미한 연속 스킵 방지

### 8) 재가입 우회 방지 수준
- 판단할 것: 정지/영구제재 사용자의 재가입 차단 강도
- 권장: **기기·IP·행동패턴·인증수단 조합으로 위험기반 대응**
- 유의: 과도한 오탐 방지를 위해 영구 자동차단보다 관리자 검토 병행이 안전

### 9) 미성년자 및 성인인증 예외 처리
- 판단할 것: 랜덤채팅 진입 전 인증 단계
- 권장: **성인인증 완료 사용자만 랜덤채팅 진입 허용**
- 이유: 서비스 성격상 가장 보수적으로 가는 편이 안전

### 10) 보존기간 세분화
- 판단할 것: 일반 대화, 신고 대화, 결제/분쟁 대화를 같은 기간으로 둘지
- 권장:
  - 일반 랜덤채팅: 목적 달성 후 단기 보존 후 파기
  - 신고/분쟁 관련 대화: 사건 종료 후 내부 기준에 따라 별도 보관
  - 전자상거래 법정기록: 법정 보존기간 준수

## 이번 코드 반영 방향
- 랜덤채팅 규칙에 중복신고 방지 정책 추가
- 신고 즉시 상대 자동 차단 정책 추가
- 허위 신고 제재 정책 필드 추가
- 메시지 자가 삭제 허용 시간 필드 추가(기본 5분)
- 관리자 신고 처리 시 허위신고 판정이면 신고자 누적 제재 반영

## 최종 권장 결정안
- 자가 삭제 허용 시간: **5분**
- 삭제 표시 방식: **상대방에게는 삭제 표기 유지, 관리자 보관소는 원문 보관**
- 허위 신고 제재: **3/5/10/15 단계형**
- 랜덤채팅 매체 범위: **텍스트만 우선 허용**
- 관리자 열람 권한: **grade 1 전용 유지**
- 차단 해제: **즉시 허용 + 로그 보관**
