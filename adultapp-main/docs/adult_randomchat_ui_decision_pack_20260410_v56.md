- 관리자모드는 조회 전용 안내를 명시

## 현재 랜덤채팅 조건 반영값
- 성별 조건: 무관 / 남 / 여 / 기타
- 성별 매칭 규칙:
  - 무관: 전체 허용
  - 남: 남만 매칭
  - 여: 여만 매칭
  - 기타: 무관, 남, 여, 기타 포함 전체 허용
- 연령 조건: 20세 ~ 99세 최소/최대 선택
- 지역 조건: 무관 / 같은 지역 우선 / 거리기반
- 거리기반 오차 허용범위: 없음
- 매칭 재시도: 4회
- 최대 탐색 시간: 5분
- 채팅방 유지시간: 20분 고정
- 종료 처리: 최근 종료 목록 유지
- 자동 종료 임박 알림: 3분 전, 1분 전
- 신고 결과: 완전 비공개
- 관리자 권한: 조회 전용

## 추가로 결심이 필요한 항목
1. 최근 종료 목록 보관 시간
- 예: 24시간 / 72시간 / 7일
- 권장: 72시간

2. 종료된 채팅 목록 표시 정보 범위
- 최소 표시: 상대 닉네임, 종료시각, 카테고리
- 권장: 내용 미노출 + 메타정보만 유지

3. 목록 정렬 기준
- 선택지: 진행 중 우선 / 최근 생성순 / 최근 메시지순
- 권장: 진행 중 우선 후 최근 생성순

4. 신고 후 즉시 목록 숨김 여부
- 선택지: 즉시 숨김 / 신고 후 유지 / 관리자 처리 후 숨김
- 권장: 즉시 숨김

5. 차단 해제 후 재매칭 쿨다운 여부
- 선택지: 즉시 재매칭 / 10분 제외 / 24시간 제외
- 권장: 즉시 재매칭 유지, 신고 건은 관리자 기준 적용

6. 최근 종료 목록 재입장 허용 여부
- 선택지: 허용 / 비허용
- 권장: 비허용

7. 관리자 감사로그 보관 기간
- 선택지: 6개월 / 1년 / 3년
- 권장: 1년
