# adultapp 정책 업데이트 정리 (2026-04-11)

- 인증사: 1차 오픈은 PASS 기반 통합 연동으로 시작하고, 카카오는 로그인 편의 수단으로만 사용
- 운영키 전환: 클로즈드 베타 동안은 테스트키 사용, 본인확인 실연동·콜백 검증·서버 저장·장애 알림·운영 점검표 완료 후 공개 베타 직전에 운영키 전환
- 미성년 처리: 미성년자는 로그인 불가 및 접속 차단, 최소 식별값과 차단 이력만 분쟁 대응 범위에서 일정 기간 보관
- 관리자 예외: 최상위 관리자만 예외 검토 가능, 감사로그를 남기고 건별 승인으로만 허용
- 재동의 정책: 필수 문서 버전이 바뀐 경우 기존 동의 사용자에게 7일 유예 후 재동의 강제

## 재동의 유예 설명
- 의미: 이미 가입했고 기존 필수 문서에 동의한 사용자에게, 문서 버전 변경 직후 서비스를 즉시 차단하지 않고 7일 동안 새 버전에 다시 동의할 시간을 주는 정책
- 대상: 기존 가입 사용자 중 필수 문서의 구버전에 동의한 사용자
- 비대상: 신규 가입자, 기존에도 필수 동의가 누락된 사용자

## 추가로 먼저 조치할 항목
1. PASS 실제 공급사 계약 및 운영키 발급
2. 콜백 서명 검증과 오류 코드 매핑 확정
3. 미성년 차단 계정 보존기간 정확한 일수 및 파기 절차 확정
4. 최상위 관리자 예외 승인 화면 또는 내부 승인 절차 문서화
5. 재동의 유예 종료 후 제한할 기능 범위 확정
6. 장애 알림 채널(Slack/메일/SMS)과 운영 점검표 양식 확정
